# 📊 학교 안전사고 데이터 분석 프로젝트

## 📌 프로젝트 개요
본 프로젝트는 2021~2025년 학교 안전사고 데이터를 기반으로  
사고 발생 특성과 보상 데이터를 분석하여  
사고 예방 및 정책 개선 방향을 도출하는 것을 목표로 합니다.

---
